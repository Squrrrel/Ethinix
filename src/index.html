Ethinix Source Code

To get the source code go onto Github For Windows and download the appliaction. After it has downloaded and installed open 'Git Shell' and follow the steps below.

Step one: Clone the Repository.

~ git clone https://www.github.com/Squrrrel/Ethinix/

Step two: Wait for Git Shell to clone the repository onto your local hard drive.

Cloning into 'Ethinix'...

remote: Counting objects: 927, done.

remote: Compressing objects: 100% (792/792), done.

Receiving objects: [Percentage Done]% ([Files Cloned]/927), [Total Size]MiB | [Download Speed]KiB/s

Resolving deltas: 100% (145/145), done.

Checking connectivity... done.

Step three: Find the local directory of the repository.

Default Directory for the folder is 'C:\Users\%USERNAME%\Documents\Github\Ethinix'.

If you saved it under a different name, then it is still located in Documents but in a different folder name.
